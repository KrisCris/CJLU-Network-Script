# CJLU-Network-Script
CJLU（计量）校园网 自动拨号 脚本

之前自己搓的挂路由器上用。。。有需要的可以自取。。

顺便。。不知道为啥用openwrt固件连校园网的时候。。路由器隔三差五就会重启。。。执行一下这个`ethtool -K eth0 tx off rx off
`就好了。。
## 2023.03.15更新
by 小鱼飘飘

为脚本适配了新版网络认证页面，并优化了断网检测模式（可能会更加有效，但是一定更快），检测时间较原始方案大幅减少

为路由器定时任务适配了重复运行检测，现在只需要简单的定时任务反复调用文件即可

如果需要重复运行判断且需要自动运行，请在路由器开机运行脚本中加入`rm /var/var_running`命令
